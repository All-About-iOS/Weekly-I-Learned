## 🚩 How To Study
- WIL Repo의 {본인} 폴더를 생성합니다.
- 주제 선정은 **기술과 관련한 자유 주제(CS, iOS 등)** 이며, 글은 **매 주 2개 이상** 작성합니다.
- 학습한 내용은 노션, 블로그, git 등을 이용해 자유로운 형식으로 정리하고, 해당 내용을 본인 폴더에 게시합니다.
